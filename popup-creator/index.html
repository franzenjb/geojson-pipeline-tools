How to get the JSON:
  1. Navigate to your ArcGIS FeatureServer URL in a browser
  2. Add ?f=json to the end of the URL (e.g., .../FeatureServer/0?f=json)
  3. Copy the entire JSON response
  4. Paste it into the textarea above

Note: If the service requires authentication, you'll need to use this manual method after logging into ArcGIS Online/Portal.

üìñ How to Use

  1. Enter your ArcGIS REST service URL
  2. Click "Fetch Fields from Service" to load all available fields
  3. Select which fields to include in your pop-up (checkboxes)
  4. Optionally group fields into sections using the grouping field
  5. Enter a title for your pop-up
  6. Click "Generate Pop-up HTML" to create the template
  7. Copy the HTML and paste it into ArcGIS Map Viewer or Experience Builder

Note: The generated HTML uses inline CSS matching American Red Cross color standards (#c8102e) and Economist-style typography.
